# ADR-014: SL/TP Planning Lines

**Status:** Implemented | **Date:** 2026-03-24

## Context
Traders need to visually plan stop-loss and take-profit levels before placing orders.

## Decision
Dashed horizontal lines rendered on chart. Red for SL, green for TP. Shows price label and distance from current price. Accessible via Trading menu → Set SL/TP Line, or drawing mode.

## Consequences
- Pro: Visual order planning without broker connection
- Pro: P&L preview at each level
